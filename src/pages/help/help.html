Internet Banking
To access Internet Banking via the application a registration is required.
Steps to follow prior to using the Mobile app
Note: You should use a Personal Computer (PC) when you log in for the very first time to your Internet Banking account. Steps to follow during your first log in,
1. Log in with your Log In ID & the Password provided by the Bank.
2. Follow the instructions to set up the secret questions.
3. Change the temporary password provided by the Bank.
4. Now you could log in to the Mobile application at any given time.
Use Facilities
It To use the following facilities, You should login via a Personal Computer (These are not available via the Mobile application)
1. Reset password activation through the secret questions.
2. Changing of secret questions.
3. Amending of personal information.
4. Uploading a secure Image.
5. Self registration of Bill payments.
6. Fund transfers to Third party accounts under Unregistered account.
